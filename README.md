# FIRO Planner（公開ビルド）

このリポジトリは **FIRO Planner** のビルド済み静的サイト（PWA）を GitHub Pages で配信するための公開リポジトリです。ソースコードと企画ドキュメントは別の非公開リポジトリで管理しています。

- 公開URL: https://worldthroughthelens245-droid.github.io/firo-planner-app/
- 内容: 資産の記録・整理と「考え方」の言語化（メンタル面の自己コーチング）を支援する**管理・教育アプリ**
- データはすべて利用者の端末内（ブラウザ）にのみ保存され、サーバーには送信されません
